# 🏢 Condominium Management API

API REST para gestão de condomínios desenvolvida com Node.js, Express e PostgreSQL.

## 🚀 Tecnologias

- Node.js
- Express
- PostgreSQL
- Prisma ORM
- Swagger

## 📌 Funcionalidades

- Cadastro de usuários
- Gestão de áreas comuns (amenities)
- Sistema de reservas
- Integração entre módulos
