# 🏢 Condominium Management API

REST API para gerenciamento de condomínios utilizando Node.js, PostgreSQL, Prisma ORM e JWT.

## 🚀 Tecnologias

- Node.js
- Express
- PostgreSQL
- Prisma ORM
- Swagger

## 📌 Funcionalidades

- Cadastro de usuários
- Gestão de áreas comuns (amenities)
- Sistema de reservas
- Integração entre módulos
